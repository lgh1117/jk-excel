# jk-excel
居于poi，做excel解析，自动识别excel或csv、tsv文件格式，单sheet最大生成百万数据在50秒以内，内存15m左右。

#个人csdn
[个人CSDN](https://blog.csdn.net/lgh1117)

##特性
   支持csv、tsv、excel所有版本
